# 🌐 OSOS - Página Web Oficial

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![ASP.NET Core](https://img.shields.io/badge/ASP.NET_Core-8.0-purple)](https://dotnet.microsoft.com/)
[![Render](https://img.shields.io/badge/Render-Deployed-green)](https://render.com)

Página web oficial del juego **OSOS - Rhythm Game**. Incluye ranking global en tiempo real, información del juego y enlaces de descarga.

---

## 🔗 **Sitio web**

[https://osos-website.onrender.com](https://osos-website.onrender.com)

---

## 🎯 **Características**

- ✅ Ranking global en tiempo real (actualiza cada 30 segundos)
- ✅ Música de fondo
- ✅ Diseño responsive (funciona en móvil)
- ✅ Galería de GIFs animados
- ✅ Botones de descarga (Google Drive, MediaFire)
- ✅ Información de controles del juego

---

## 🚀 **Tecnologías utilizadas**

| Tecnología | Uso |
|------------|-----|
| **ASP.NET Core MVC** | Backend web |
| **HTML5 / CSS3** | Estructura y estilos |
| **JavaScript (Vanilla)** | Actualización dinámica del ranking |
| **Firebase Realtime Database** | Almacenar y leer ranking |
| **Docker** | Contenedor para despliegue |
| **Render** | Hosting en la nube |
